
# CICIDS Dataset Download Instructions

This folder should contain the CICIDS 2017 dataset.

## Download Steps:
1. Visit: https://www.unb.ca/cic/datasets/ids-2017.html
2. Download the CSV files
3. Place them in this directory

## Dataset Info:
- Real network traffic captures
- Contains benign and malicious traffic
- Used in academic research

## Files Expected:
- Monday-WorkingHours.pcap_ISCX.csv
- Tuesday-WorkingHours.pcap_ISCX.csv
- etc.

If dataset is too large, use NSL-KDD as alternative:
https://www.unb.ca/cic/datasets/nsl.html
